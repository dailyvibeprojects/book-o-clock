# book-o-clock
Count the time before bedtime in books

## Why
Our kid loves being read to before bed. We often agree on a books count based on how late it is. But every other night we end up repeating "It's too late, we cannot read one more book"

## What it does?
It shows an analogue clock and the remaining time before bedtime, measured in books. You can adjust the bedtime hour as well as the average duration of the books in minutes.
